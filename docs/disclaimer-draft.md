## 1. 产品性质声明

**「天枢城」（以下简称"本工具"）是一款用于辅助文字角色扮演（Role-Play）创作的纯前端静态界面工具。** 本工具的主要功能为：协助用户管理虚构世界观设定、剧情上下文、提示词模块、AI 对话注入逻辑等创作素材。

**本工具不提供、不运营、不控制任何人工智能模型、算法、数据或 API 服务。** 您所看到的所有由 AI 生成的内容，均为您本人通过本工具发起的、与第三方人工智能服务商（如 Anthropic、OpenAI、Google、智谱、阿里云、深度求索等）API 接口的直接交互结果，与开发者无关。

本工具不内置任何具体剧情内容、角色人设或叙事文本；所有创作内容均为用户自行编写、导入或与第三方 AI 共同生成。
